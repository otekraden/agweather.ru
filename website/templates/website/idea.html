The Website Idea:

At this point in time, the site was created for educational purposes as a portfolio project.
The idea of the site came with the following question:
What happens if you save the weather forecast data and then compare it with the actual weather that has occurred?
Thus, it will be possible to graphically evaluate which of the services is most accurate in certain conditions.
The task of covering the whole world and all available public data of weather sites is difficult for me at the moment.
Therefore, if you would like to see the location you are interested in on the site, then you need to link it manually.
The necessary tools for it are presented in the navigation bar on the left in the Contribution section.
